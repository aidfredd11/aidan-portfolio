Areas of Interests

Machine Learning

More specifically deep artificial neural networks, and their use for solving vastly complex problems is of great interest to me. In a recent academic project, I used the Python framework PyTorch and its built-in AlexNet pretained model to extract features from a dataset of 50,000 training images.

Web Development

The internet is how you are able to read this very sentence. This site was built using HTML, CSS, and vanilla JavaScript, however, I have used both Angular and React for web development. Past projects of mine include using the MEAN stack to develop a library checkout system, and using the MERN stack to develop an ecommerce application, complete with user authentication.

Game Programming

The feeling of seeing a virtual world that you have created in action, is one like no other. Using Unity and C#, I have programmed both games and developer tools. Three notable projects of mine are a maze running game, a rock-climbing game, and a custom Unity editor tool for character customization.
